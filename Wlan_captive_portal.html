Herzlich Willkommen im W-LAN Heidelberg4you!

... oder doch nicht?

Sie sind hier in einem fake W-LAN-Netzwerk gelandet.

SCHOCK

Warum?

Weil sich Ihr Smartphone/Laptop/internetfähiges Gerät automatisch mit einem W-LAN-Netz verbindet ohne vorher zu überprüfen, ob es das richtige bzw. ein sicheres Netz ist.

Wo ist dabei das Problem?

Wenn Ihr Gerät mit einem "falschen" Netzwerk verbunden ist, können Sie, wenn Sie die URL-Adresse von einer Website eingeben, auf einer anderen Website landen. Das ist nicht nur nervig, sondern kann auch gefährlich werden: z.B. wenn Sie dadurch eine verbotene Website aufrufen oder im Darknet unterwegs sind.

Wie kann ich mich davor schützen?

Wenn Sie W-LAN nicht benötigen, können Sie es ganz ausschalten. Dann kann sich Ihr internetfähiges Gerät nicht mit einem "falschen" Netzwerk verbinden, wenn Sie z. B. in der Stadt unterwegs sind. Ansonsten können sie die Funktion ausstellen, dass sich Ihr internetfähiges Gerät automatisch mit einem Netzwerk verbindet. Dann können Sie manuell entscheiden, mit welchem W-LAN-Netzwerk sich Ihr Gerät verbindet.

Wenn Sie trotzdem nicht wissen, ob sich Ihr Gerät mit dem richtigen W-LAN verbunden hat, können Sie auf das Kürzel https achten, um zu überprüfen, ob die Website sicher ist.
